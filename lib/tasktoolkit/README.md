# node-mf-tasktoolkit-dll
=========

    Node.js Jukebox DLL

## Overview
=======

This is a Node.js module that provides a DLL (Dynamic Link Library) to bind to multiple technical features exposed by Moment Factory.
Included features are:
  - Tasktoolkit: A set of functions to bind your software to Moment Factory's Control Center application.
  - Pulse: Publish the state of your software through MQTT.


## Installation
=======

To install this module, run the following command after downloading a release of this repository:

```bash
# npm install
```
